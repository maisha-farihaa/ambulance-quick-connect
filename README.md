 🔗 Live App

Description:
This is a web-based ambulance service application.

The system is designed to help users quickly connect with ambulance services during emergencies.


##  Features
- Quick ambulance request system  
- Emergency contact interface  
- Simple and user-friendly UI  
- Fast access to available ambulance services  
- Mobile responsive design
